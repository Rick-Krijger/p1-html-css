Waarom noem je de eerste HTML-pagina; index.html?

Omdat het de basis is voor de rest van de website, het skelet, daarna komen CSS en JavaScript pas.

Met welke HTML versie werken we nu?

Versie 5.

Met welke CSS versie werken we nu?

Versie 3.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Sommige browsers intrepeteren code net iets anders, oudere browserversies ondersteunen bijvoorbeeld niet de nieuwste versie van HTML of CSS.
